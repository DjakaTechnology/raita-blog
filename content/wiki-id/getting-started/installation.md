## Persyaratan Sistem

| Platform | Minimum |
|---|---|
| Windows | Windows 10 64-bit atau lebih baru |
| macOS | macOS 11 (Big Sur) atau lebih baru, Intel atau Apple Silicon |
| Linux | Saat ini tidak didukung |

---

## Unduh

Unduh installer terbaru dari [area anggota Raita](https://www.raita.ai/member). Pilih versi untuk sistem operasi Anda:

- **Windows** — installer `.msi`
- **macOS (Apple Silicon)** — `.dmg` untuk Mac M1/M2/M3
- **macOS (Intel)** — `.dmg` untuk Mac Intel

---

## Instal

**Windows:**
1. Jalankan file `.msi`
2. Ikuti petunjuk installer
3. Raita akan muncul di menu Start Anda

**macOS:**
1. Buka file `.dmg`
2. Seret Raita ke folder Applications Anda
3. Pada peluncuran pertama, macOS mungkin menampilkan peringatan keamanan — buka **System Settings → Privacy & Security** dan klik **Open Anyway** (pada macOS 12 dan lebih awal: **System Preferences → Privacy & Security**)

---

## Peluncuran Pertama

Pada peluncuran pertama, Raita akan:
1. Membuat database lokal di folder data aplikasi Anda
2. Menampilkan layar login

Lanjutkan ke [Setup Akun](account-setup.md) untuk mendaftar atau masuk.
