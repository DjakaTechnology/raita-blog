## System Requirements

| Platform | Minimum |
|---|---|
| Windows | Windows 10 64-bit or later |
| macOS | macOS 11 (Big Sur) or later, Intel or Apple Silicon |
| Linux | Not currently supported |

---

## Download

Download the latest installer from the [Raita member area](https://www.raita.ai/member). Choose the version for your operating system:

- **Windows** — `.msi` installer
- **macOS (Apple Silicon)** — `.dmg` for M1/M2/M3 Macs
- **macOS (Intel)** — `.dmg` for Intel Macs

---

## Install

**Windows:**
1. Run the `.msi` file
2. Follow the installer prompts
3. Raita will appear in your Start menu

**macOS:**
1. Open the `.dmg` file
2. Drag Raita to your Applications folder
3. On first launch, macOS may show a security warning — go to **System Settings → Privacy & Security** and click **Open Anyway** (on macOS 12 and earlier: **System Preferences → Privacy & Security**)

---

## First Launch

On first launch, Raita will:
1. Create a local database in your app data folder
2. Show the login screen

Continue to [Account Setup](account-setup.md) to register or log in.
